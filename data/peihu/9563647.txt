Method and system for generating a state profile report. Disclosed are systems and methods relating to profiling software and configuration information associated with one or more entities. In various implementations, software and configuration information associated with one or more entities may be retrieved. The retrieved software and configuration information may provide state information associated with the one or more entities. The retrieved information may be processed to identify one or more changes in the software and configuration information that is associated with the one or more entities. A report may be generated based on the retrieved information. The report may display identified changes in a state for at least one entity based on the changes in the software and configuration information that were identified during the processing.
