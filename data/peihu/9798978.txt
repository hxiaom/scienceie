Hybrid data architecture for use within a travel industry optimized cognitive environment. A data architecture for use within a cognitive information processing system environment comprising: a plurality of data sources, the plurality of data sources comprising a public data source and a private data source, the public data source comprising publicly available travel information, the private data source comprising privately managed, company specific travel information; and, a cognitive data management module, the cognitive data management module accessing information from the plurality of data sources and providing the information to an inference and learning system.
