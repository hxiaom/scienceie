Enhanced system and method for control of robotic devices. A system for robotic device control and data acquisition, comprising a robotic device control system adapted to receive sensor-based data comprising physical object information, the sensor-based data being received from a plurality of sources, each source comprising at least a physical sensor in communication with the robotic device control system computer via a communications network, an enhanced robotic device control application adapted to receive sensor-based data from the robotic device control system computer and manipulate the sensor-based data to produce enhanced data, and a method for robotic device control and data acquisition.
