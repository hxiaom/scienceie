Warehouse data processing system and method. A warehouse data processing system including a handheld terminal and a database server, wherein the handheld terminal is connected with the database server, the handheld terminal is used to scan goods information and/or stock location codes, acquire stock data including the goods information and/or the stock location codes from the database server, and perform warehouse data processing on the stock data according to the scanned goods information and/or stock location codes. The warehouse data processing is achieved in a mobile and portable manner by using the handheld terminal to scan goods information and/or stock location codes and performing warehouse data processing on the stock data according to the scanned goods information and/or stock location codes.
